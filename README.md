# ADMN

A tensorflow implementation of ["Attention-based Adaptive Memory Network for Recommendation with Review and Rating"](https://ieeexplore.ieee.org/abstract/document/9099276)

## Requirement
 - python >= 3.6
 - tensorflow = 1.x (1.7)
 - numpy
 - pandas
 - ntlk

## Training

    python main.py
